Quick Installation Guide

Get Deepest Seeker running in under a minute.

  1. Download the ZIP: Click any "Download" button on this page. Save the `deepest_seek_0520.zip` file to your computer.
  2. Unzip the File: Find the downloaded ZIP file and extract its contents. You'll get a folder (e.g., "deepest-seeker-extension").
  3. Open Chrome Extensions:
    • Type `chrome://extensions` in your Chrome address bar and press Enter.
    • Alternatively, click the three dots (⋮) in Chrome -> Extensions -> Manage Extensions.
  4. Enable Developer Mode: In the top right corner of the Extensions page, toggle on "Developer mode".

    Image showing where to find the developer mode toggle.

  5. Load Unpacked: Click the "Load unpacked" button that appears (usually on the top left).

    Image showing the 'Load unpacked' button.

  6. Select the Folder: In the file dialog, navigate to and select the folder you unzipped in Step 2 (e.g., "deepest-seeker-extension"). Do NOT select the ZIP file itself.
  7. Done! Deepest Seeker should now appear in your list of extensions. Pin it to your toolbar for easy access!
